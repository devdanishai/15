1. install multipass
```bash
sudo snap install multipass

multipass launch --name ubuntu-server --disk 20G --memory 2G
multipass launch --name ubuntu-server1 --disk 5G --memory 2G

multipass shell ubuntu-server

multipass list

multipass stop ubuntu-server

multipass start ubuntu-server

multipass shell ubuntu-server # go inside vm if running
multipass info ubuntu-server # get ip address

# delete vm
multipass delete ubuntu-server
multipass purge

```
1. passwd of root:
```bash
sudo passwd root
su - root
sudo passwd ubuntu



2. run small commands
```bash
whoami
hostname
hostname -I
lsb_release -a
df -h
free -h
ip a
```

3. install packages
```bash
sudo apt update && sudo apt install nginx -y

sudo systemctl status nginx

# visit http://10.208.246.230/ 
# you see nginx page
```

4. Next Skill: Firewall & SSH Security (LAB 5)
- Allow Web Traffic + SSH in firewall
```bash 
sudo ufw allow 'Nginx HTTP'
sudo ufw allow OpenSSH
sudo ufw enable
sudo ufw status
```
- Harden SSH (Prevent brute-force password attacks)
```bash
sudo apt install fail2ban -y
sudo systemctl enable --now fail2ban
sudo systemctl status fail2ban
```
- Create a new admin login
```bash
sudo adduser admin1
sudo usermod -aG sudo admin1 #give sudo previllege to admin1
su - admin1
sudo -i -u ubuntu #switch user w/o passwd

```
