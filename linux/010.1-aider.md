### 1️⃣ Go to your project
cd ~/projects/myapp

### 2️⃣ Create a venv
python3 -m venv .venv

### 3️⃣ Activate it
source .venv/bin/activate

### 4️⃣ Install Aider (google-generativeai NOT needed)
pip install aider-chat

### 5️⃣ Run Aider
aider --model gemini/gemini-2.0-flash-exp
